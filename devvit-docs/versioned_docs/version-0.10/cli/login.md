## `devvit login`

Log in to Devvit via reddit.com

```
USAGE
  $ devvit login [--copy-paste]

FLAGS
  --copy-paste  If present, the user will copy-paste their code from the browser, rather than use localhost.

DESCRIPTION
  Log in to Devvit via reddit.com
```
